# Contributing to 1D QT Ideal Solver

## Development Setup

```bash
git clone https://github.com/yourusername/1d-qt-ideal-solver.git
cd 1d-qt-ideal-solver
pip install -e ".[dev]"
```

## Code Style

```bash
black src/
```

## Submitting

1. Fork repository
2. Create feature branch
3. Make changes
4. Submit pull request
